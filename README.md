Name resolution written in plain Haskell.

This library is much less mature than <https://github.com/haskell-suite/haskell-names>.

# Supported extensions
* FunctionalDependencies
* Rank2Types
* RankNTypes
* EmptyDataDecls
* BangPatterns
* ForeignFunctionInterface
* MagicHash
